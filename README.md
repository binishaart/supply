Supply Chain Anomaly Predictor

A Flask-based web app that monitors shipment data and global news to detect potential delays, risks, or anomalies in supply chains. Provides real-time predictions and rerouting suggestions via a simple AI dashboard.

Features

Predicts shipment anomalies (mock AI for demo purposes).

Simple and responsive web UI.

Real-time feedback on shipments.

Ready for deployment with Gunicorn.

Tech Stack

Backend: Python 3.x, Flask

Frontend: HTML, CSS, JavaScript

Deployment: Gunicorn (Render / Heroku)
